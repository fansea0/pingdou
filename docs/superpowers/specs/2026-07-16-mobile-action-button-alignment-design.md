# 移动端导入导出按钮对齐设计

## 目标

让移动端底部操作栏中的“上传”和“导出图片”按钮在任意状态下均水平、垂直居中文本。

## 方案

- 仅在移动端媒体查询内，为两个底部操作按钮增加比全局 `button.primary` 更具体的选择器。
- 覆盖全局 `width: 100%`，使按钮按内容宽度布局；同时明确 `justify-content: center`。
- 保留现有导出进度层、加载状态、成功状态、可访问名称与交互逻辑。

## 不变项

- 桌面端导出按钮及其他主按钮样式不变。
- 上传、导出功能和禁用逻辑不变。

## 验证

- 新增组件测试断言两个移动端按钮具有对齐所需的局部类名。
- 运行组件测试、类型检查和生产构建。
